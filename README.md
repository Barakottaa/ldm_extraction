# Oracle Forms PL/SQL Extraction

This repository contains PL/SQL code extracted from Oracle Forms binary files (.fmb).

## Files

- **extracted_plsql_code.sql** - Initial extraction with core procedures and functions
- **additional_plsql_code.sql** - Additional procedures missed in initial extraction
- **reg2_module_plsql.sql** - REG2 module with comprehensive registration logic

## Overview

Extracted from Oracle Forms modules for a laboratory information management system (LIMS). Contains:

- Patient registration logic
- Test management procedures
- Security and authentication
- Validation and business rules
- Reporting functions
- SuperPro hardware key integration

## Total Code

Approximately 1,500+ lines of PL/SQL code across 3 files.
